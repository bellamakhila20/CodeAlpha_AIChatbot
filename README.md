# AI Chatbot

## Description

AI Chatbot is a simple Java console application that interacts with users using predefined responses. It demonstrates basic conversational logic using conditional statements and loops.

## Features

- Greeting responses
- Java-related questions
- Internship-related questions
- Rule-based conversation
- Exit command
- User-friendly interface

## Technologies Used

- Java
- Object-Oriented Programming
- Scanner Class
- Conditional Statements
- Loops

## How to Run

Compile:

```bash
javac AIChatbot.java
```

Run:

```bash
java AIChatbot
```
